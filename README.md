PickTrue
------------

强迫症怎么能看到喜欢的画师不收图呢？

哪个画师不需要自己的Visual Library呢？


# 下载

+ [windows-x64](https://github.com/winkidney/PickTrue/releases)
+ [macOS](https://github.com/winkidney/PickTrue/releases)

Linux请直接下载并安装Python项目即可，有打包需求再提
